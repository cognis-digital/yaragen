# Security Policy

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

- Email: `security@cognis.digital`
- Include: affected tool + version, impact, and reproduction steps or a PoC.
- We aim to acknowledge within 3 business days and to provide a remediation
  timeline after triage.

Coordinated disclosure is appreciated; we will credit reporters who wish to be
named once a fix is released.

## Scope & responsible use

The Cognis Neural Suite includes **dual-use security tooling**. Use it only against
systems, data, and identities you own or are explicitly authorized in writing to
test. Misuse is solely your responsibility and may violate law.

## Supported versions

Until a tool reaches `v1.0`, only the latest released version receives security
fixes.
